## Aim: 
To Design a wireframe diagram for a generic application or website using Balsamiq

## Algorithm:
1.Start the application.

2.Display the Splash Screen with logo and options Login and Sign Up.

3.If the user selects Login, go to the Login screen.

4.If the user selects Sign Up, go to the Sign-Up screen.

5.On the Login/Sign-Up screen, the user enters details and confirms to proceed.

6.Validate user credentials.

7.If valid, navigate to the Home Screen.

8.Display the Home Screen showing playlists, new releases, and music options.

9.When a user selects a song, open the Music Player Screen.

10.Allow play, pause, next, and previous actions on the Music Player.

11.If the user clicks the contact icon, open the Contact Screen.

12.Display contact details and social media links.

13.If the user wants to return, navigate back to the Home Screen.

14.When the user exits, close the app.
